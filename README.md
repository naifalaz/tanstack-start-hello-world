# SkyLaunch

SkyLaunch is an outdoor gear inventory and discovery app built as a portfolio-ready MVP. It allows users to browse products, search gear, and explore a polished client-side experience backed by Supabase.

## Live Demo
- Production: https://your-vercel-app.vercel.app

## Tech Stack
- TypeScript
- React 19
- TanStack Router
- Tailwind CSS v4
- Supabase
- Vite
- Vercel

## Features
- Product browsing and seeded demo data
- Search across realistic gear records
- Local-only AI and Hybrid search during development
- Health endpoint for deployment checks
- Basic unit and end-to-end testing
