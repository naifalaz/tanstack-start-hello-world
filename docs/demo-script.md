# SkyLaunch Demo Script

## 1. Opening
"SkyLaunch is an outdoor gear discovery MVP designed to show a polished product catalog experience with realistic seeded data, search, and production-ready deployment practices."

## 2. Show the Main Experience
1. Open the home page
2. Point out the clean product grid and realistic images
3. Use the search feature to find a product category such as tents or backpacks
4. Open a product detail or highlight product information if your UI supports it

## 3. Explain Admin Controls
- Mention that add, edit, and delete controls are intentionally toggled instead of always visible
- Explain that this keeps the UI cleaner for demo users

## 4. Explain Local-Only AI Features
"AI and Hybrid search are available in local development, but hidden in production because Vercel does not run the local Ollama embedding workflow used during development."

## 5. Technical Highlights
- Built with React 19, TypeScript, TanStack Router, Tailwind CSS v4, Supabase, and Vercel
- Includes a health endpoint and smoke test workflow
- Includes basic unit and end-to-end tests

## 6. Closing
"The result is a portfolio-ready MVP that is not only functional, but also documented, testable, and demo-ready."
